Restoring Purchases

If you purchased premium content and it is not showing as unlocked, open Pick Panic! and use the Restore Purchases option. Make sure you are signed into the same Apple ID that was used for the original purchase.

Reporting a Problem

When contacting support, please include:

  • Your device model, if known
  • Your iOS version, if known
  • What you were doing when the issue happened
  • Screenshots, if helpful
